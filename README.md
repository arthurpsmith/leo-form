# MyCrossref

## Setup - required
```
npm install
```

### I want to...

#### run a hot-reloading dev server
```docker-compose up```

or

```
npm run serve
```

### compile and minify for production
```
npm run build
```

### run unit tests
```
npm run test:unit
```

### run end-to-end tests
```
npm run test:e2e
```

or (with GUI)

```
./run-cypress.sh -g
```

or (headless)

```
./run-cypress.sh
```

### lint and fix syntax
```
npm run lint
```
